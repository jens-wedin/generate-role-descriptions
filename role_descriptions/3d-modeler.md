# 3D Modeler
## Description
Specializes in building 3D characters, environments, and objects based on concept art.

## Org Level
IC

## Medium
Digital

## Industry
Gaming, Architecture, Tech

## Responsibilities
-  sculpt and model characters and props
- Create UV maps and textures
- Ensure topology is suitable for animation
- Work within polygon count limits

## Skills
- ZBrush
- Topology
- Hard Surface Modeling
- Anatomy
