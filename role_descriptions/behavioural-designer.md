# Behavioural Designer
## Description
Applies psychology and behavioral economics to design products that influence user behavior.

## Org Level
IC

## Medium
Service

## Industry
Tech, Health, Finance

## Responsibilities
- Analyze user behaviors and motivations
- Design interventions to change behavior
- Conduct A/B testing
- Apply behavioral frameworks

## Skills
- Psychology
- Data Analysis
- Experimentation
- UX Design
